AI_API bool AiTest ( )

Runs all unit tests.

This function runs a battery of unit tests – detailed internal consistency checks that are meant to detect bugs in the API. The Arnold software is required to pass all unit tests before shipping. In the unlikely event of this function failing, please report to the developers along with the text that is printed to stdout.

Returns
true if all tests passed succesfully
